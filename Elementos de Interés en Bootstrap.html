Custom jumbotron

Using a series of utilities, you can create this jumbotron, just like the one in previous versions of Bootstrap. Check out the examples below for how you can remix and restyle it to your liking.

Modal Window

Un jumbotron es un titulo con letras relativamente grandes, y una pequeña descripción de un concepto que se quiere llamar la atención, e inmediatamente abajo un boton para ir a algún lugar. El siguiente jumbotron fue tomado, replicado de https://getbootstrap.com/docs/5.3/examples/jumbotron/. Lo que se hizo fue simplemente copiar el código en la ventana de inspección, si uno quiere replicar una parte de una página web puede utilizar este método, siempre y cuando esa página esté utilizando Bootstrap.
